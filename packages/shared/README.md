# @ku0/shared

`@ku0/shared` contains repo-wide utilities and small UI-facing helpers shared by
the active HugeCode app surfaces.

Current exports include:

- product about content and external URL helpers
- runtime gateway environment and browser helpers
- shared feature flags, type guards, and small utility modules
- lightweight shared UI primitives such as chat and navigation helpers

This package is part of the active workspace graph and is published from the
TypeScript source entrypoints declared in [package.json](./package.json).
